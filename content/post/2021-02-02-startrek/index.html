Amazingly enough, the top bigram is actually meaningful - ‚Äúnumber one‚Äù, by Picard, refers to the ship‚Äôs commander, William Riker. But most of the rest of the top 10 bigrams are pretty boring - ‚Äúthis is‚Äù, ‚Äúof the‚Äù, ‚Äúit is‚Äù, ‚Äúto the‚Äù. Some of these bigrams consist of two stopwords together.

Stopwords do not add much meaning to a sentence, and we can usually safely remove them. Since we have bigrams, two words together, here, it‚Äôs not a matter of just anti-joining on a stopwords data set. Instead, I‚Äôll take the approach of removing a bigram if both of the words are stopwords.

From the {tidytext} package, I‚Äôm using the ‚Äúsnowball‚Äù stopword lexicon - this will allow for more conservative stopword removal, since it contains way less words than the other two lexicons:

stop_words %>%

tng_stopwords <- stop_words %>%
  filter(lexicon == "snowball") %>%
  pull(word)

To see if both words in a bigram are stopwords, I‚Äôll split the bigram into two columns, and test if each is a stopword (yes I know how to use across() wowww):

bigrams <- bigrams %>%

We could just look at each character‚Äôs top bigrams, but I want to see what makes them them; what makes them unique - what if all of the characters have the same top bigrams? Boring!

Because I‚Äôm only human, I‚Äôll go ahead and just copy Julia Silge‚Äôs excellent blog post on {tidylo} and use weighted log odds, a method of comparing features (bigrams) across some set (characters).

Since this is ~tidy text mining~, it‚Äôs super easy to get the top 5 bigrams for each character, by the weighted log odds (no I don‚Äôt know what it is but it compares them, OK? I‚Äôll remove the x-axis labels anyways)

library(tidylo)

tng_tidylo <- bigrams %>%
  bind_log_odds(character, bigram, n) %>%
  group_by(character) %>%
  top_n(5, log_odds_weighted) %>%
  ungroup()
